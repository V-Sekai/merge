# README

1. Fetch this repository
1. Ensure git email and name are setup.
1. On Windows use msys2.
1. Login to ssh key
1. run `./update_godot_v_sekai.sh` to release the branch.
1. run `./release_godot_v_sekai.sh` to update the production branch.
